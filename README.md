----------------

Fluento_Dyslexia_Detector is designed to be an all-in-one solution for sentiment analysis, text processing, and visualization. The application is built using popular Python libraries such as Streamlit, PIL, TextBlob, Pandas, and SpeechRecognition. With Fluento_Dyslexia_Detector, you can effortlessly analyze large text datasets, generate reports, and even convert text to speech.

The application is divided into three primary modules:

1. **Text Analysis**: This module is responsible for analyzing text data, identifying sentiment, and extracting keywords. Fluento_Dyslexia_Detector uses the TextBlob library to perform sentiment analysis and extract relevant features from the text.
2. **Visualization**: This module provides a range of visualization options, including word clouds, bar charts, and pie charts, to help users visualize their data and gain insights.
3. **Speech Output**: This module uses the SpeechRecognition and pyttsx3 libraries to convert text to speech, providing an audio output of the analyzed text.
